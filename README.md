# Gzong

Gzong，一个以Go为基础的简单web框架



## Examples

```sh
# 编译
$ go build Gzong
```

```sh
# 启动服务
$ go run Gzong/hello.go
```

```sh
# 测试服务
$ curl localhost:8080/testhello
```
